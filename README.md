
## HOW TO USE
Clone this repo
  ```bash
  git clone https://github.com/silkyland/30-mins-react-redux {yourfoldername}
  ```
#### Using Yarn
 ```bash
  cd {yourfoldername}
  yarn 
```
#### Using NPM
``` bash
  cd {yourfoldername}
  npm install
```

### Serve your project
```bash
  yarn start 
  or
  npm run start
```
